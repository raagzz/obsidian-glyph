# Glyph

A clean, readability-focused Obsidian theme — slightly technical, but not a full terminal/monospace look.

## Status

Early / in development. Not yet submitted to the community theme store.

## Installation (manual, for now)

1. Clone or download this repo.
2. Copy it into `<vault>/.obsidian/themes/Glyph/`.
3. Enable it in Settings → Appearance → Themes.
